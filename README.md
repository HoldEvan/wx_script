# WeiMi-微秘

基于 MonkeyCode 平台的微信插件，BeanShell 脚本构建，Xposed 模块环境运行。

## 功能

| 模块 | 功能说明 |
|---------|-------------|
| 每日六十秒 | 每天获取新闻简报 |
| 摸鱼日报 | 每日摸鱼资讯 |
| 历史今天 | 历史上的今天 |
| 表情合成 | 图片表情合成 |
| 小人举牌 | 举牌文字生成 |
| TTS 播报 | 消息文字转语音朗读 |
| 自动回复 | AI 自动回复消息 |
| 专属铃声 | 消息通知/音视频来电专属铃声 |
| 定时消息 | 定时发送消息（支持多时间点） |
| 天气预报 | 实时天气查询 |
| 实时金价 | 黄金价格查询 |
| IP 查询 | IP 地址归属查询 |

## 入口

- 启动命令：`/微秘`（可自定义）
- 所有功能均可在主面板中开启/关闭

## 技术栈

- 语言：Java / BeanShell
- 平台：MonkeyCode
- 环境：Xposed 模块
